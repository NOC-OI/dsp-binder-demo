# Lightweight Data Science Platform demo with Mybinder

This repository contains a lightweight demonstration of the Data Science Platform's Docker images using Mybinder. 
It should contain enough software for some basic data analysis tasks or running an introductory Carpentries workshop. 

## Launch the Demo

Press the "Launch Binder" button above to launch. 

## What Software is Included?

This is image includes:

* Python 3.11/Jupyter 5.2.1
* Linux command line utilities including Git, GCC, Make, Man (and all the man pages), Nano, Vim and Emacs
* Matplotlib and Numpy libraries
* JWM Desktop
* Gnuplot
